  # 🎬 Bias-Aware OTT Content Performance Predictor

## 📌 Problem
OTT platforms rely on engagement metrics like watch time and completion rate.  
However, these are biased toward mobile users, under-representing Smart TV audiences (family, rural viewers).  

This leads to unfair content decisions.

---

## 💡 Solution
We built a **bias-aware ML system** that:
- Applies **device-based bias correction**
- Adjusts under-represented Smart TV data
- Predicts content performance more fairly

---

## ⚙️ Features
- Bias correction layer
- Engagement score prediction
- Random Forest ML model
- Interactive Streamlit dashboard

---

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
